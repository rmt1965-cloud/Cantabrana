# Cantabrana: Retazos de Vida

Experiencia audiovisual inmersiva que rescata la memoria sonora de Cantabrana a través de sus puertas y jotas.

## 📱 Funcionamiento
Cada escaneo de un código QR genera una experiencia única basada en la **aleatoriedad**:
* **Momento de Gloria**: 14 segundos de materia viva (IA) que brota o cae.
* **Texturas**: Agua marina, vino, miel, alquitrán, magma, hielo y musgo.
* **Textos**: Selección aleatoria de fragmentos poéticos sobre los linajes y la piedra de Cantabrana.
* **Pergamino**: Aviso de sensibilidad con tinta dinámica (sangre o carbón).

## 🛠️ Instrucciones de Producción
1. Colocar los audios en `/jotas` siguiendo el formato `Nombre--00.mp3`.
2. Ejecutar `produccion_maestra.py` para:
    * Actualizar el inventario digital (`galeria.json`).
    * Generar las tarjetas de impresión en `/QRS_IMPRIMIR`.
3. Subir los cambios a GitHub para activar la web.

## 📂 Estructura de Activos
* `/assets/pergamino`: Fondos de pergamino antiguos.
* `/assets/sellos`: Sellos de cera y heráldica.
* `/fotos/arcos_puertas`: Fotografías nítidas de entradas centenarias.
